验证 ClipPartner 从登录、权限、后台运营、分发者准入、授权、素材、任务、发布核验、风控、结算、钱包、接口和部署检查的完整功能闭环，确保线上代码保持可演示、可协作、可部署。

## 4. 通用前置条件

- 已执行 `npm install`。
- 本地前端可通过 `npm run dev` 启动。
- 如测试 Worker API，本地 Worker 可通过 `npm run worker:dev` 启动。
- 浏览器清理或保留 localStorage 时要在测试记录中注明，因为当前系统支持本地 fallback 状态。
- 若连接远程 API，确认 `.env.local` 中 `NEXT_PUBLIC_API_BASE_URL` 指向正确环境。
